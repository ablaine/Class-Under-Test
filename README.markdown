Class Under Test
================

Requirements
------------
- Java 6
- Maven (to build)

Installation
------------

1. Checkout the project
2. Run `mvn clean install`
3. Include the dependency

Include in your pom.xml:

    <dependency>
    	<groupId>com.ablaine</groupId>
    	<artifactId>class-under-test-core</artifactId>
    	<version>0.1.0-SNAPSHOT</version>
    </dependency>
