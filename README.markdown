Class Under Test
================

Requirements
------------
- Java 6
- Maven (to build)

Installation
------------

- Checkout the project
- Run mvn clean install
- Include
`
<dependency>
	<groupId>com.ablaine</groupId>
	<artifactId>class-under-test-core</artifactId>
	<version>0.1.0-SNAPSHOT</version>
</dependency>
`
